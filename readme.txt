== Description ==

This plugin is designed to remove version number printing from wp_head to help protect your wordpress site.

== Installation ==

Installing "RM Version" can be done either by searching for "RM Version" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
2. Activate the plugin through the 'Plugins' menu in WordPress
